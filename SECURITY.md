# Security Policy

## Supported Versions

Security fixes are provided for the latest published version of Filament Tutorials.
Users should upgrade to the newest release before reporting an issue that may
already have been resolved.

## Reporting a Vulnerability

Please do not disclose suspected vulnerabilities in public issues, discussions,
or pull requests.

Report vulnerabilities privately through
[GitHub Security Advisories](https://github.com/CoringaWc/filament-tutorials/security/advisories/new).
If private reporting is unavailable, email `argemirodiaseng@gmail.com` with the
affected version, reproduction steps, impact, and any proposed mitigation.

You should receive an acknowledgment within three business days. Confirmed
vulnerabilities will be coordinated privately until a fix and release guidance
are available.
